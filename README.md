# SQL CRUD

## Repository Structure

### 1. **create/**
- `create_tables.sql` — create example tables  
- `insert_data.sql` — insert sample data into tables  

### 2. **read/**
- `select_all.sql` — select all records from a table  
- `select_where.sql` — filter records using WHERE  
- `select_join.sql` — join two tables  

### 3. **update/**
- `update_single_record.sql` — update a single record  
- `update_multiple_records.sql` — update multiple records with conditions  

### 4. **delete/**
- `delete_single_record.sql` — delete a single record  
- `delete_multiple_records.sql` — delete multiple records based on condition
